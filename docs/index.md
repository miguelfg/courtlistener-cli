# CourtListener CLI

`courtlistener-cli` is a Python command line client for the CourtListener REST API. It covers opinions, search, courts, dockets, RECAP documents, parties, attorneys, judges, oral arguments, financial disclosures, alerts, citations, tags, and batch requests.

Use these docs as the CLI manual:

- Start with [Installation](installation.md) and [Authentication](authentication.md).
- Use [Quickstart](quickstart.md) for the first commands to run.
- Use [Command Reference](command-reference.md) for generated Click help.
- Use [Examples](examples/index.md) for practical command recipes.

Most commands accept `--format json`; many list/search commands also support `csv` and `xlsx` exports through `--format` and `--output`.
