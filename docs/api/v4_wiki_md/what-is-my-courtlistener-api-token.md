# What is my CourtListener API Token?

**Source:** [](docs/raw/wiki.free.law/c/courtlistener/help/api/rest/faqs/what-is-my-courtlistener-api-token.html)

---

What is my CourtListener API Token? Every CourtListener account has an API token for use with our APIs. To find your API token, visit your profile where you will find your API token and API usage metrics. Simply copy your token from there and you can use it with our legal APIs.
