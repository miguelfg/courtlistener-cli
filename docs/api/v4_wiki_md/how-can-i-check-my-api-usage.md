# How can I check my API usage?

**Source:** [](docs/raw/wiki.free.law/c/courtlistener/help/api/rest/faqs/how-can-i-check-my-api-usage.html)

---

How can I check my API usage? There is a page on your profile that lists your API usage for the last few days: https://www.courtlistener.com/profile/api-usage/ For now it doesn't show your overall API usage, and it is not API-accessible, but we aim to address these needs soon.
